# auto-optimizer-v1

Projet de travail (mode script) pour l’auto optimizer v1.

## Démarrage rapide (Windows)

```bash
py -m venv .venv
.venv\Scripts\activate
pip install -r requirements.txt
python src\main.py
```

## Secrets

- Ne pas commiter de fichier `.env`.
- Utiliser `.env.example` comme modèle.
